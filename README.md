# back-sOmnistack11
Aplicação dos aprendizados da semana Omnistack 11.0

*Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

   - Nodejs;
   - celebrate;
   - cors;
   - cross-env;
   - express;
   - helmet;
   - knex;
   - sqlite3;


Run the commands on root folder:

    npm i (to push and install all dependencies)


To start project:

    npm start

    npm run dev

   localhost:3333
